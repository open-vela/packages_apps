## 小米穿戴设备 Apps
主要用于存放【小米穿戴设备】的相关应用，其中包括系统应用示例和三方应用示例。目前主要技术栈为[Xiaomi Vela QuickJS](https://iot.mi.com/vela/quickapp/zh/guide/)。

### 系统应用：
手表上的一方应用，主要有Launcher、设置等。
#### 1. Launcher 
Launcher 应用旨在为用户提供便捷、快速的应用访问体验。通过简洁直观的界面设计，用户可以轻松找到并启动他们喜欢的应用程序。
